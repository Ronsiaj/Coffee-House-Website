# Coffee-House-Website
Responsive Coffee House single-page website built with ReactJS and Reactstrap.

A modern **responsive single-page Coffee House website** built using **ReactJS and Reactstrap**.  
The website showcases a clean café style UI with sections like hero banner, features, opening hours, menu highlight, testimonials, and contact.

---

## 🚀 Features

- Responsive single-page design
- Modern coffee shop UI
- Mobile-friendly layout
- Hero section with call-to-action
- Features section
- Why Choose Us section
- Opening Hours display
- Coffee menu highlights
- Testimonials section
- Social media footer
- Clean and simple navigation

---

## 🛠️ Technologies Used

- ReactJS
- Reactstrap
- CSS3
- HTML5
- React Icons

## ▶️ How to Run the Project

1. Clone the repository

2. Navigate to the project folder

    cd coffee-house-website


3. Install dependencies

    npm install


4. Run the development server

    npm start
